## 简介

Admin 是一个参考[Vue Vben Admin](./http://doc.vvbin.cn/) 的中后台模版。

## 特性

- **最新技术栈**：使用 Vue3/vite2 等前端前沿技术开发
- **TypeScript**: 应用程序级 JavaScript 的语言
- **主题**：可简易配置的主题
- **组件** 二次封装了多个常用的组件

## 安装使用

- 获取项目代码

```bash
git clone https://github.com/wudunkai/vue-admin.git
```

- 安装依赖

```bash
cd vue-admin

pnpm install

```

- 运行

```bash
pnpm run dev:web
pnpm run dev:server
```

- 打包

```bash
pnpm run build:web
```
